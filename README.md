# Automatizaciones con Python

Colección de scripts en Python orientados a automatizar tareas cotidianas
de una persona que trabaja en el área administrativa.

## Objetivo

El propósito principal de este repositorio **no es tener scripts perfectos**,
sino aprender Python de forma práctica resolviendo problemas reales del día a día.

## ¿Qué hay aquí?

Cada archivo `.py` resuelve una tarea puntual como:

- Leer y generar archivos Excel
- Monitorear correos electrónicos
- Procesar y organizar información
- Enviar reportes automáticamente

## Tecnologías usadas

- Python 3
- `openpyxl` — manejo de archivos Excel
- `imaplib` — lectura de correos
- `smtplib` — envío de correos

## Cómo ejecutar un script

**1. Crea y activa el entorno virtual:**
```bash
python3 -m venv venv
source venv/bin/activate.fish  # Fish shell
```

**2. Instala las dependencias:**
```bash
pip install -r requirements.txt
```

**3. Ejecuta el script:**
```bash
python3 nombre_script.py
```
